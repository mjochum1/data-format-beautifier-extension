# JSON / YAML / JSONL Beautifier

A Firefox extension that beautifies structured data files like `.json`, `.jsonl`, `.yaml`, and `.xml`. Features include:

- Syntax highlighting
- Line numbering
- Copy-to-clipboard
- Toggle to interpret `\\n`, `\\t`, etc.

## Usage

1. Open a `.json`, `.yaml`, or `.jsonl` file in Firefox (via `file://`)
2. The extension automatically beautifies and highlights it
3. Use the "Copy" button or "Show Raw \\n" toggle

## Build & Install (Development)

1. Open `about:debugging#/runtime/this-firefox`
2. Click "Load Temporary Add-on"
3. Select the `manifest.json` file
